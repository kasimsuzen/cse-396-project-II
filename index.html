This repository is for CSE396 course of Gebze Technical University Computer engineering Faculty.

Project Description

We created a traffic/road assistant system. Which will recognize road lines, emergency lines, traffic density, speed detection, traffic sign recognition and if the vehicle goes under bridge or driving in a tunnel.

Project Modules

Our project has two modules as hardware and image processing software using different methods such as: 1. Machine Learning in Traffic sign and vehicle recognition 2. Haar cascade in vehicle detection 3. Optic flow in speed detection 4. Color modifying and thresholding in tunnel drive 5. Houghline, canny edge detection algorithm and angle calculation for line detection
